# 🐘 Postgres Performance Tuning

Master PostgreSQL optimization. Practical scripts for indexing, vacuuming, and configuration tuning. By **Ahmar Hussain** (**Stackaura**).

## 🛠️ Scripts
- `check_bloat.sql`: Identify table and index bloat.
- `tune_conf.sh`: Auto-tune `postgresql.conf` based on system RAM.
- `slow_query_logger.sql`: Setup advanced slow query logging.
